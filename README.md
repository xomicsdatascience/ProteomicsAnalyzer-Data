# ProteomicsAnalyzer-Data
Data to accompany the proteomics-analyzer example vibe coded application.

"filtered_anti_non.csv" is a subset of the HepG2 +/- antibiotics proteomics data presented by Movassaghi and Meyer biorxiv 2025

"sc_skm_leidens_reindexed_T.csv" is the single muscle fiber proteomics data from Momenzadeh et al Journal of the American Society for Mass Spectrometry 2025
